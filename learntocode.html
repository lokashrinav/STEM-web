Learn to Code!
There are several ways you can learn how to code, from tutorials all the way to courses.

The following tiles contains some popular course platforms and websites where you can learn how to code.
W3 Schools
W3 Schools is a website where you can learn how to code online. w3 Schools offers free practice questions and certification exams where you can test your knowledge. It teaches HTML, CSS, JavaScript, PHP, SQL, and more!

Click on the following link to go to W3 Schools: https://www.w3schools.com/
Full Stack open
Full Stack open is an open-source project contributed to by several people where you can learn full stack javascript. In this course, you can learn the essentials of modern web development, along with React, Redux, Node.js, MonogoDB, and TypeScript.

Click on the following link to go to Full Stack open: https://fullstackopen.com/en/
FreeCodeCamp
FreeCodeCamp is a website where you can learn different coding aspects, like responsive web design, front end development, javascript algorithms, and more! It also gives free certifications and is a great way to start coding with its guides!

Click on the following link to go to FreeCodeCamp: https://www.freecodecamp.org/
Code.org
Code.org is a nonprofit that helps students at many schools learn the fundamentals of computer science and coding. Students of all ages can benefit from using Code.org and is a great way to start coding as a beginner.

Click on the following link to go to FreeCodeCamp: https://code.org/
