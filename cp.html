Competitive Programming
Don't get confused between Competitive Programming and Hackathons! Competitive Programming is typically associated with finding a solution to a programming algorithmic problem, while Hackathons are project-based. These coding problems could be super easy or terrifyingly scary! Competitive Programming is super fun and something you should try out at least once if you decide to become a coder!

The following tiles are platforms where you can start competitive programming:
CodeForces
Codeforces is a website that hosts virtual competitive programming contests. It was founded by Mirzayanov and is one of the most famous competitive programming sites of all time. In addition, it offers a competitive programming ratings, from Newbie to Legendary Grandmaster, making the competitive programming journey even more competitive! It hosts hundreds of contests along with thousands of challenging coding problems.
USACO
USACO is one of the most famous competitive programming platforms for high schoolers. USACO stands for the United States of America Computer Olympiad and can help greatly in the college applications process. Although this competition is targeted towards high schoolers, non-high schoolers can still compete (However, you won't be able to join the IOI USA Team). USACO also offers one of the best training materials for competitive programming created by previous participants called the USACO Guide. Make sure to check it out.
Code Jam
Code Jam is one of the most famous coding competitions in the world and is run by Google. You start in the preliminaries, then go to Round 1, Round 2, Round 3, and finally World Finals. It is said that Google will offer you interviews if you do well in Code Jam. Code Jam is only a select time in the year so make sure to check when registration opens.
Hacker Cup
The Hacker Cup is a competitive programming competitions run by Facebook. It's also known as the Meta Hacker Cup. It typicall starts in August and goes all the way until December. In this annual competition, you'll have to solve algorithmic coding problems in a set amount of time. This competition can help you get interview calls from Facebook.
ICPC
The ICPC, or the International Collegiate Programming Contest is an algorithmic programming contest for college students, typically for undergraduates. Even if you're not a college student, you can practice using past ICPC material. However, if you can, try to participate in this competitions. It is without a doubt one of the most famous competitions around.
